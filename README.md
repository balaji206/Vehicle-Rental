# 🚗 Vehicle Rental System

A full-stack **Vehicle Rental Web Application** that allows users to browse, search, and rent vehicles easily.  
The project is built using **React.js for the frontend** and **Spring Boot for the backend**, providing a modern UI with a powerful REST API.

---

## 📌 Project Overview

The **Vehicle Rental System** enables users to explore available vehicles, view vehicle details, and book vehicles for rent.  
It provides a seamless experience for both customers and administrators to manage rentals efficiently.

---

## 🛠️ Tech Stack

### Frontend
- ⚛️ React.js
- 🎨 CSS / Tailwind CSS
- 🔗 Axios
- 🌐 React Router

### Backend
- ☕ Spring Boot
- 🗄️ MySQL / H2 Database
- 🔐 Spring Security (Optional)
- 📦 Maven

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/vehicle-rental-system.git
cd vehicle-rental-system

```

### 2️⃣ Backend Setup (Spring Boot)

Navigate to backend folder

```bash
cd backend
```
Run the application

```bash
./mvnw spring-boot:run
```
Backend runs on

```bash
http://localhost:8080
```

### 3️⃣ Frontend Setup (React)

Navigate to frontend folder
```bash
cd frontend
```

Install dependencies
```bash
npm install
```
Run the application
```bash
npm start
```
Frontend runs on
```bash
http://localhost:3000
```
